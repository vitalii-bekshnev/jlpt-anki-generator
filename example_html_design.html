Design Principles

  • Inline CSS Only: All styles are inline to work within Anki's constraints
  • Mobile Responsive: Designs adapt to different screen sizes
  • Color Coding: Different colors for JLPT levels (N5=blue, N4=green, N3=orange, N2=red, N1=purple)
  • Visual Hierarchy: Clear distinction between word/reading/meanings/examples
  • No External Dependencies: All self-contained except Noto Sans JP font

Usage in CSV

When generating CSV files for Anki import:

  • Wrap HTML content in double quotes
  • Escape internal quotes properly
  • Newlines within quoted fields are preserved
  • Test import in Anki after generation
